### To Run
`Python 3.12` version에서 테스트되었습니다.  

가상환경 생성 및 활성화 (필요하면 실행)
```shell
python3 -m venv .venv
source .venv/bin/activate
```

필요한 패키지 설치
```shell
pip install -r requirements.txt
```

서버 실행
```shell
# 최초 실행 시에만 필요
python3 manage.py makemigrations
python3 manage.py migrate
# 서버 실행
python3 manage.py runserver
```
